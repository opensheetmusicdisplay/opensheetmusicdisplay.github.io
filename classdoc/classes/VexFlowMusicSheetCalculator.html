    ProtectedskyBottomLineCache

    skyBottomLineCache: Map<string, { bottom: number[]; sky: number[] }> = ...

    Lazy rendering: cache of computed sky/bottom lines, keyed per staff line by its system's measure range + staff index. A growing-prefix batch re-runs the (expensive) skyline pass over the whole prefix; this lets stable interior systems reuse the lines computed in an earlier batch instead of re-measuring them. Only used when EngravingRules.LazyConsistentGraphic; cleared per lazy session.

    calculateElongationFactorFromStaffEntries

    • calculateElongationFactorFromStaffEntries(
          staffEntries: GraphicalStaffEntry[],
          oldMinimumStaffEntriesWidth: number,
          elongationFactorForMeasureWidth: number,
          measureNumber: number,
          previousLyricOverflows?: number[],
          previousChordOverflows?: number[],
      ): {
          factor: number;
          lastChordEntryDict: { [i: number]: any };
          lastLyricEntryDict: { [i: number]: any };
      }

      Parameters

      • staffEntries: GraphicalStaffEntry[]
      • oldMinimumStaffEntriesWidth: number
      • elongationFactorForMeasureWidth: number
      • measureNumber: number
      • OptionalpreviousLyricOverflows: number[]

        Per-verse-index array ([verseIndex]) holding how far the previous measure's last lyric extends past its bar line into this measure (in pre-elongation units, +dashSpace if mid-word). Used to seed lastLyricEntryDict so the first lyric in this measure is forced to leave clearance from the overhang.

      • OptionalpreviousChordOverflows: number[]

        Same as previousLyricOverflows but for chord symbols.

      Returns {
          factor: number;
          lastChordEntryDict: { [i: number]: any };
          lastLyricEntryDict: { [i: number]: any };
      }

      • factor: regular elongation factor from within-measure spacing constraints (subject to MaximumLyricsElongationFactor cap by the caller).
        • lastLyricEntryDict / lastChordEntryDict: final state of the per-verse last-entry dicts after processing. The caller uses these (with the post-cap measure width) to compute the overflows passed into the next measure's call.

    ProtectedcalculateMeasureXLayout

    • calculateMeasureXLayout(measures: GraphicalMeasure[]): number

      Calculates the x layout of the staff entries within the staff measures belonging to one source measure. All staff entries are x-aligned throughout all vertically aligned staff measures. This method is called within calculateXLayout. The staff entries are aligned with minimum needed x distances. The MinimumStaffEntriesWidth of every measure will be set - needed for system building. Prepares the VexFlow formatter for later formatting Does not calculate measure width from lyrics (which is called from MusicSheetCalculator)

      Parameters

      • measures: GraphicalMeasure[]

      Returns number

      the minimum required x width of the source measure (=list of staff measures)
