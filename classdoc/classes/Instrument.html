Transpose

Transpose: number = 0
  • Defined in MusicalScore/Instrument.ts:23

Transposition halftones for this instrument only. This is additive to osmd.Sheet.Transpose (MusicSheet). osmd.TransposeCaculator needs to be defined/created for this to take effect. (just set it with new TransposeCalculator()) You need to call osmd.updateGraphic() before the next render() (assuming this is set after load()).
